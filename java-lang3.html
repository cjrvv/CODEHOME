Java Comments
Comments are used in the program to both clarify it and make it simple for the programmer to distinguish between all of its parts.

There's a two ways to create a comment first the Single-line Comments and the other one is the Multi-line Comments.

Single-line Comments

You can put a Single-line comments in your program by using the two slashes //.

Any text you put after the double slashes will be ignored by Java.

public class MyProgram {

public static void main(String args[]) {

// You can comment here!

System.out.println("Hello World!");

}

}

You can comment in single line only.

public class MyProgram {

public static void main(String args[]) {

System.out.println("Hello World!"); // And also you can comment here!

}

}

And also you can add a comment on the line of your code.

Multi-line Comments

You can put a comments in a multiple lines in your program starts with one slash and one asterisk /* and ends with one asterisk and one slash */.

Any text you put between these two /* and */ will be ignored by Java.

public class MyProgram {

public static void main(String args[]) {

/* You can comment here
with multiple lines it only ends using this -> */

System.out.println("Hello World!");

}

}
