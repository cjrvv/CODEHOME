Java Syntax
In every programming languages syntax are impormant it's like a rule that we need to follow, so that the computer easily understand our program. If our syntax are incorrect, the program will not compile. Let's say you created the following code in a Java file called MyProgram.java to display "Hello World!" on the screen:


public class MyProgram {

public static void main(String args[]) {

System.out.println("Hello World!");

}

}

Output
After you compile your program, you'll see the output of your program to the console like the display below.

Hello World!

Explanation
In your program you used System.out.println() which can be used to print an argument passed to it. Inside the parenthesis you need to put a double quotation like this ("") to identify it as a "String" variable.

public class MyProgram {

public static void main(String args[]) {

System.out.println("Hello World!");

}

}
