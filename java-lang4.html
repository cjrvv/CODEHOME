Java Variables
Variables it's like a storage this is where you can store data values in a particular variable.

There are different types of variables:

String - can store text, for example "MY NAME" inside the parenthesis double quotations are necessary.

int - can only store a whole number, for example 26 and also we can add negative like -2004 as long as our value is a whole number.

char - can only store a single character it's either capital or small letter. We use single quotation on this variable. For example 'a' and 'A'.

boolean - can only stores true or false.

float - can only stores a decimal number.

Different ways to declare variables
Here an examples on how to declare different variables

String OnlyText = "Today is my Birthday!";

int OnlyWhole = 14;

char OnlySingle = 'C';

boolean TAndF = true;

float DeciNum = 14.08;
